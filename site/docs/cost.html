cost

Snapshot Cost Timeline + cold-tier transition candidates. Pluggable cost provider; defaults to S3 Standard list-price rates.

What it does

Walks the snapshot history, summing per-snapshot added-bytes minus removed-bytes from the snapshot summary. Emits per-snapshot cumulative GB, monthly USD at the configured rate, and a tier-candidate flag for snapshots older than the cold-tier cutoff.

Example

$ iceberg-sentry cost \
    --catalog rest --rest http://catalog:8181 \
    --table finance.transactions \
    --cold-tier-days 90
~ cost timeline

Flags

FlagDefaultDescription
--tablerequiredFully qualified table id.
--rate0Override $/GB/month for the standard class (0 = use provider default).
--providers3-standardInformational provider name surfaced in output.
--cold-tier-days90Snapshots older than this are flagged as cold-tier candidates.
--formattexttext · json

Built-in rates (list price)

standard$0.023 / GB-mo
infrequent-access$0.0125 / GB-mo
glacier-instant$0.004 / GB-mo
glacier-deep$0.00099 / GB-mo

All figures are labelled as estimates. Override --rate for negotiated discounts. Phase 4 will register additional providers (Azure ADLS Gen2, GCS).
