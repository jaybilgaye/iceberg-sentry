Policy as Code — sentry.yaml

Version-controlled thresholds, scoped by namespace glob. Apply with --policy sentry.yaml on any command that supports it.

A complete example

version: "1.0"
default_catalog: glue

policies:
  - name: finance-compliance
    target_namespace: "finance.*"
    max_snapshot_age: 30d
    min_file_size_mb: 128
    max_manifest_files: 500
    max_partition_skew_percent: 20
    pii_scan: true
    fail_on_orphans: false
    fail_on_pii_untagged: true
    delete_file_ratio_warn: 0.10
    delete_file_ratio_fail: 0.25
    write_pattern: auto
    min_health_score: 75

  - name: raw-zone-relaxed
    target_namespace: "raw.*"
    max_snapshot_age: 7d
    min_file_size_mb: 32
    write_pattern: streaming
    min_health_score: 50
    pii_scan: true
    fail_on_pii_untagged: true

Field reference

default_catalogstringCatalog name applied when CLI lacks --catalog.
policieslistOne per namespace pattern.
policies[].namestringLabel for diagnostics.
policies[].target_namespaceglobMatches namespaces. Supports * and trailing .*.
min_file_size_mbintOverrides file_size threshold.
max_manifest_filesintWarn threshold for manifest_density (crit is 2×).
max_snapshot_agedurationWarn threshold for snapshot dim. 14d, 30d, 12h.
delete_file_ratio_warnfloatWarn threshold for delete_amplification.
delete_file_ratio_failfloatCritical threshold.
write_patternenumauto · streaming · batch. Forces the classifier.
min_health_scoreintCI gate: fail if composite drops below.
pii_scanboolOperational hint consumed by orchestration.
fail_on_pii_untaggedboolIf true, surfaces exit code 3.
fail_on_orphansboolOperational hint consumed by orchestration.

Namespace matching

Patterns are matched against the table's namespace, not the table name. First-match-wins, in document order.

  • "*" or "" — match every namespace.
  • "finance" — exact match.
  • "finance.*" — match finance and any namespace starting finance. (Iceberg supports nested namespaces).

Duration syntax

max_snapshot_age accepts the Go time.Duration syntax plus a "days" extension:

  • 30d, 7d
  • 12h, 90m, 3600s

Usage in CLI commands

$ iceberg-sentry audit --policy sentry.yaml --namespace finance --fail-on warn

Combine with --fail-on warn in CI so warnings (which the policy intentionally raises before critical) surface as build failures.
